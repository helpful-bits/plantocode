### D. Mechanisms (5 modules)

1. **Code‑Aware Discovery** — From 1,000 files to the **true impact surface** (deps, patterns, non‑obvious links).
2. **Multi‑Model Planning** — "Council of LLMs" → compare philosophies → resolve conflicts → one coherent blueprint.
3. **Integrated Terminal & CLI** — Run Claude Code / Codex CLI / Aider inside PlanToCode. **Stage prompts** (with voice dictation), **approve**, then send. Keep a **traceable transcript**.
4. **Safe Handoff** — Read‑only plan/approval flows; export to Claude Code / Cursor / Codex CLI if you prefer external tools.
5. **Transparency & Control** — Live token/costs, editable prompts, local sessions.

### E. 3‑Step Flow

1. **Describe** — Type, *dictate*, or **record** a 2‑min repro/brief.
2. **Discover** — File + dependency discovery builds minimal, sufficient context.
3. **Plan → Execute** — Generate/merge plans across models; **run in the integrated terminal** or export to your agent.
